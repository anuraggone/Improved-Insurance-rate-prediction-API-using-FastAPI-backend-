## Improvements in This Version

This is the improved version of the **Insurance Rate Prediction API** built using a FastAPI backend.

In this version, the following enhancements have been implemented:

### 1. Improved Project Structure
- Refactored the project into a modular structure
- Implemented proper module imports
- Applied separation of concerns for better maintainability and scalability

### 2. Enhanced API Response
- The API now returns the prediction result
- Includes prediction score
- Provides outcome probability for better interpretability

### 3. Structured Data Validation
- Added a dedicated `schema` folder
- Separated data validation logic from business logic
- Improved code organization, validation, and reliability
